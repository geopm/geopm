
geopm::CPUActivityAgent(3) -- agent for selecting CPU frequency based on CPU compute activity

--------

#include `<geopm/CPUActivityAgent.hpp> <https://github.com/geopm/geopm/blob/dev/src/CPUActivityAgent.hpp>`\
