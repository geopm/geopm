
Getting Started
===============

The GEOPM project's original use case is to enable traditional High
Performance Computing (HPC) workloads to be more energy efficient.  This goal
has several requirements that enable a broader set of applications to leverage
the GEOPM software.

The following pages outline the primary use cases of GEOPM and community-maintained publications that discuss this project, respectively.
